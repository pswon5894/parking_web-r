# 로그인 버튼 클릭 시 모달 창으로 변경하기

이 문서는 기존 `Login.js` 컴포넌트를 수정하여, '로그인' 버튼을 클릭했을 때 모달 창이 나타나고 그 안에서 사용자가 아이디와 비밀번호를 입력하도록 변경하는 방법을 안내합니다.

## 2. `client/src/components/Login.js` 파일 수정

`Login.js` 컴포넌트를 모달 상태를 관리하고, 버튼 클릭 시 모달을 띄우도록 수정합니다. 아래의 전체 코드로 `client/src/components/Login.js` 파일의 내용을 교체하세요.

```javascript
// client/src/components/Login.js
import React, { useState } from 'react';
import ReactDOM from 'react-dom';
import { useAuth } from '../context/AuthContext';
import '../App.css'; // Import CSS for modal styles

function Login() {
  const [username, setUsername] = useState('');
  const [password, setPassword] = useState('');
  const [error, setError] = useState('');
  const [isModalOpen, setIsModalOpen] = useState(false);
  const { user, loading, login, logout } = useAuth();

  const handleLoginSuccess = () => {
    setIsModalOpen(false); // Close modal on successful login
    setUsername(''); // Reset fields
    setPassword('');
    setError('');
  };

  const handleSubmit = async (e) => {
    e.preventDefault();
    setError('');
    try {
      await login(username, password);
      handleLoginSuccess();
    } catch (err) {
      setError('로그인에 실패했습니다. 아이디와 비밀번호를 확인하세요.');
    }
  };

  const openModal = () => setIsModalOpen(true);
  const closeModal = () => {
    setIsModalOpen(false);
    setError(''); // Reset error when closing
  };

  if (loading) {
    return <div>로딩 중...</div>;
  }

  // If user is logged in, show welcome message and logout button
  if (user) {
    return (
      <div style={{ padding: '10px', textAlign: 'center' }}>
        <span>환영합니다, {user.username}님!</span>
        <button onClick={logout} style={{ marginLeft: '15px' }}>로그아웃</button>
      </div>
    );
  }

  // If not logged in, show Login button
  return (
    <>
      <div style={{ padding: '10px', textAlign: 'center' }}>
        <button onClick={openModal}>로그인</button>
      </div>

      {isModalOpen && ReactDOM.createPortal(
        <div className="modal-overlay" onClick={closeModal}>
          <div className="modal-content" onClick={e => e.stopPropagation()}>
            <button className="modal-close-btn" onClick={closeModal}>&times;</button>
            <h2>로그인</h2>
            <form onSubmit={handleSubmit} className="login-form">
              <div>
                <label>사용자 이름:</label>
                <input
                  type="text"
                  value={username}
                  onChange={(e) => setUsername(e.target.value)}
                  required
                  autoFocus
                />
              </div>
              <div>
                <label>비밀번호:</label>
                <input
                  type="password"
                  value={password}
                  onChange={(e) => setPassword(e.target.value)}
                  required
                />
              </div>
              {error && <p style={{ color: 'red', textAlign: 'center' }}>{error}</p>}
              <button type="submit">로그인</button>
            </form>
          </div>
        </div>,
        document.body
      )}
    </>
  );
}

export default Login;
```

## 변경 사항 설명

1.  **모달 상태 추가**: `isModalOpen` 상태를 추가하여 모달 창의 열림/닫힘 상태를 관리합니다.
2.  **UI 변경**:
    *   사용자가 로그인하지 않았을 경우, 이제는 전체 폼 대신 '로그인' 버튼만 표시됩니다.
    *   이 버튼을 클릭하면 `isModalOpen` 상태가 `true`가 되어 모달 창이 화면에 나타납니다.
3.  **모달 구현**:
    *   `ReactDOM.createPortal`을 사용하여 `ImageModal.js`와 동일한 방식으로 모달을 렌더링합니다. 이렇게 하면 다른 요소들과의 스타일에 영향을 받지 않고 최상단에 표시됩니다.
    *   모달의 배경(overlay)을 클릭하면 모달이 닫히도록 설정했습니다.
    *   모달 내부에 기존의 로그인 폼을 그대로 배치하고, 닫기(X) 버튼을 추가했습니다.
4.  **로그인 성공 처리**: 로그인에 성공하면 `handleLoginSuccess` 함수가 호출되어 모달 창을 닫고 입력 필드를 초기화합니다.
5.  **스타일 적용**: `App.css`에 추가한 클래스들을 `className`으로 지정하여 모달의 디자인을 개선했습니다.

이 변경으로 사용자 경험이 향상되며, 로그인 인터페이스가 필요할 때만 깔끔하게 표시됩니다.
